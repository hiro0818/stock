# 投資リクエスト(動作確認用サンプル)

## 投資方針

- **目的**: 中長期の資産形成 + 学習(運用しながら経済・企業分析を学ぶ)
- **投資期間**: 3〜5年(短期売買しない)
- **リスク許容度**: 中庸(年率 -20% までの含み損は耐えられる)
- **既存ポートフォリオ**: 預金中心、リスク資産はインデックス投信が少額のみ
- **除外条件**: タバコ・武器産業は除外

## 検討したい銘柄

主にテック大型 + 日本の輸出企業を中心に、まず基礎を固めたい。

### 米国株
- **AAPL**(Apple): iPhone エコシステム、自社株買いの常連
- **MSFT**(Microsoft): クラウド + AI 二刀流
- **GOOGL**(Alphabet): 検索独占 + AI 投資が重い
- **NVDA**(NVIDIA): AI ブームの中心、PER 高水準

### 日本株
- **7203.T**(トヨタ): 円安耐性、HEV 強い、EV は遅れている
- **6758.T**(ソニーG): エンタメ + 半導体センサー
- **9984.T**(ソフトバンクG): Vision Fund のリスク、Arm 上場後の見通し

## 知りたいこと

1. 各銘柄のバリュエーション(割安/割高、過去レンジでの位置)
2. 主要な強みと、現時点の懸念点
3. ポートフォリオに組み入れる場合の目安比率
4. 撤退・損切のトリガーとなる事象

## 注意点

- 本リクエストはあくまで**学習目的**であり、最終的な売買判断は自分が行う。
- 一次資料(決算短信、IR 資料)で必ず裏取りする。
