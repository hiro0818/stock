## 詳細実行ログ

### サイクル 7(LightGBM ML)─ 最も貢献度の大きい改善
- **Plan**: 過去リターン(1/5/20/60/252日)+ ボラティリティ(20/60日)+ MA 比率 + RSI + MACD + 出来高比 + 52週レンジ位置 + マクロ 12 指標の 20 日リターンを特徴量に、勾配ブースティング回帰で 21 営業日後のログリターンを予測
- **Do**: `tools/predict_ml.py::predict_lightgbm` 実装、walk_forward に統合、10 銘柄全部で実行
- **Check**: 全銘柄横断の方向当たり率 **56.2%**(2 位の linear 51.4% を 4.8pt 突き放す)、誤差 10.79%(中位)
- **Act**: 重み学習で重み 15.2% を獲得、アンサンブル方向当たり率 +1.4pt に貢献

### サイクル 6(暗号専用モデル)
- **Plan**: BTC との 90 日ベータと BTC 自身の線形外挿の積で、COIN/MSTR を予測
- **Do**: `tools/predict_ml.py::predict_crypto_linked` 実装
- **Check**: 暗号銘柄(COIN/MSTR)で誤差 20.71%/22.54%(従来モデル横並び)、ただし**全銘柄横断の誤差は 10.16% で最小**(ベータが小さい銘柄で穏当な予測になる)
- **Act**: 重み 17.1% を獲得(最大)。暗号銘柄予測の改善は限定的だが、全体安定化に貢献

### サイクル 14-32(20 サイクル重み学習)
- 学習率 0.15、誤差逆数比例ルール
- サイクル 1(均等)→ 20(収束):誤差 10.45 → **10.32**、方向 53.0 → **54.6**
- 単調収束、発振なし

## 80% 目標との距離

| | 値 | 80% との差 |
|---|---|---|
| ベースライン(v1、5 銘柄)| 51.6% | -28.4pt |
| v2 重み学習収束(5 モデル) | 53.2% | -26.8pt |
| **v3 重み学習収束(7 モデル + ML)** | **54.6%** | **-25.4pt** |

差は **-25.4pt のまま** 大きい。理由は v2 報告書の 6.2 章で説明したとおりランダムウォーク仮説。

## 結論

7 つの改善仮説 + 20 サイクル重み学習で、**実測の改善幅は誤差 -0.15pt / 方向当たり率 +1.4pt**。
これが個人ツール + 1 ヶ月ホライズン + ランダムウォーク制約下で得られる**現実的な改善幅の指標**。

「もっと劇的に改善したい」場合の道筋:
- **予測ホライズンを短く**(1 日 / 1 週間先):効率市場の隙間が大きく、精度は上がる
- **オルタナティブデータ**:衛星画像、ウォルマート駐車場、信用カード集計(各 億円規模の投資)
- **マイクロ秒アルゴ取引**:インフラ投資が個人レベルでない
- **インサイダー情報**:違法、対象外
