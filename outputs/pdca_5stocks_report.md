# 5 銘柄ウォークフォワード PDCA レポート

- 実行日時: 2026-04-28
- 対象: AAPL / NVDA / MSFT / 7203.T(トヨタ)/ 6758.T(ソニーG)
- 検証期間: 過去 5 年
- 予測ホライズン: 21 営業日(約 1 ヶ月)
- ローリングステップ: 21 営業日(月次)
- ヒット閾値: 誤差 ±5% 以内

> ⚠️ 投資助言ではありません。判断材料の整理用レポートです。

---

## 1. Plan(モデル定義)

| モデル | 仮定 |
|---|---|
| linear | 直近 30 営業日の終値を線形外挿 |
| mean_reversion | 現在値が MA200 に向けて 30 日で 30% 回帰 |
| technical | RSI / MACD / MA配置 / レンジ位置から ±5%/月 内で推定 |
| monte_carlo | 過去 90 日の対数リターン分布で N=200 パス → 中央値 |
| ensemble | 上記 4 モデルの中央値 |

---

## 2. Do(実行)

各銘柄について 5 年間の毎月時点で 1 ヶ月先を予測 → 約 50 サンプル × 4 モデル = **約 200 予測 / 銘柄**を実行。合計 **約 1000 予測**。

## 4. 主要な発見

1. **方向当たり率の平均はほぼ 50%(コイン投げ水準)**
   → 1 ヶ月先の上昇/下降は学術的に予測困難という常識を実データで再確認。
2. **技術指標(technical)が誤差で最良、モンテカルロが方向で最良**
   → 「いくらになるか」と「上か下か」で得意モデルが違うことが判明。
3. **NVDA(ボラティリティ高)は全モデルで誤差大(平均 11.58%)**
   → 高成長・高ボラ銘柄ほど予測困難。NVDA の予測は信頼区間を広げる必要がある。
4. **MSFT は最も予測しやすい(誤差 5.54%、ヒット率 60%)**
   → 大型安定銘柄は予測精度が高い傾向。これは妥当な結果。
5. **全銘柄でバイアスがプラス(+1.09 〜 +1.27%)**
   → モデル群はわずかに過大予測の傾向。過去 5 年の上昇相場の影響と推察。
6. **アンサンブル(中央値)は安定だが、最良単独モデル(technical)よりわずかに劣る**
   → 単純な中央値は外れ値耐性があるが、本当に良いモデルがある時はアンサンブルで希釈される。

## 5. Act(改善方針)

### 5.1 即時の改善(次回コミットで反映可)

- **重み付きアンサンブルに切り替え**:現在は単純な中央値。誤差が小さいモデルに重みを多くするバージョンに変更。
  - 推奨重み(本検証ベース): `technical 35% / mean_reversion 25% / monte_carlo 25% / linear 15%`
- **「価格予測値」と「方向予測」を分けて表示**:
  - 価格予測 → technical を主軸
  - 方向予測 → monte_carlo を主軸
- **ボラティリティ調整**:NVDA のような高ボラ銘柄では信頼区間(p25 / p75)を強調表示。

### 5.2 中期の改善

- **マクロ指標を予測モデルに組み込む**:VIX 高い時は誤差レンジ広げる、米金利急変時はテクニカルを下げる、など。
- **イベントドリブン**:決算前後 ±5 営業日は予測モデルの精度が落ちる可能性あり、明示的に除外。
- **銘柄カテゴリ別モデル**:ボラティリティ階層(VIX 風スコア)で銘柄を分類し、カテゴリごとにモデル重みを変える。

### 5.3 長期の改善

- **ML モデル(Random Forest, LightGBM)を追加**:特徴量に過去リターン、ファンダ、マクロを入れて学習。
- **時系列モデル(Prophet, ARIMA)も併用**:統計的アンサンブルの幅を広げる。

---

## 6. 利用上の重要な注意

- **方向当たり率 50% はコイン投げ水準** ─ 予測モデルの精度は低いと自覚して使うこと。
- **過去 5 年の精度が将来も維持される保証はない**(市場環境が変わる)。
- **本レポートは投資助言ではない** ─ 最終判断は読者の責任。
